Supported Formats
ChemKED YAML

ChemKED (Chemical Kinetics Experimental Data) is a standardized YAML format for storing experimental combustion data, particularly ignition delay measurements.

ReSpecTh XML

ReSpecTh (Repository of Spectral and Thermochemical Data) XML files are automatically converted to ChemKED format using PyKED's built-in converter.

Supported Features:
  • Ignition delay experiments
  • Shock tube and RCM data
  • Species composition with InChI/SMILES
  • Uncertainty specifications
  • Time history data (volume, pressure)
Example Structure:
file-version: 0
chemked-version: 0.4.1
file-authors:
  - name: John Doe
reference:
  doi: 10.1016/...
  journal: Combustion and Flame
  year: 2020
experiment-type: ignition delay
apparatus:
  kind: shock tube
datapoints:
  - temperature: [1200, kelvin]
    pressure: [2, bar]
    ignition-delay: [500, us]
    composition:
      kind: mole fraction
      species:
        - species-name: H2
          amount: [0.05]
